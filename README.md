# Software_club_PUMyeik

Software Club web application for the University of Computer Studies, Myeik.

## Features
- Member and admin authentication
- Member settings with profile update
- Admin dashboard for members, events, and knowledge sharing
- Blog and knowledge pages with reactions and comments
- MongoDB-backed API for persistent data

## Tech Stack
- React
- Vite
- Zustand
- Node.js
- Express
- MongoDB

## Project Structure
- `src/` frontend application
- `server/src/` backend API and database models

## Run Locally
### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
npm run dev
```

## Build
```bash
npm run build
```
